Q1 : How does Javascript work?

Answer : Javascript is a single-threaded programming language.It means execution by single line or expression.Javascript execute only one command at a time in a specific order up to down,That mean's it only go to the next line,when it finished the previous line,which mean syncronously.Although it has some asyncronous bahavior.But it work normaly serial wise or syncronously.

In together,Javascript is a interpreted language.Interpreted means It read the code line by line and execute directly.But for so quick exeqution browser used javascript engine.Google chrome used v8 engine(This engine can be different according to the browser,for example: mozila firefox used spider-monkey ).For executing javascript code in the browser v8 engine used Just In Time(JIT) compiler.Frist of all, v8 engine parse the code in abstract syntax tree.Then proccess the code by Just In Time compiler or TurboFan.Then created Optimized Machine Code and finally it display in the browser.

Q3 : What are the difference between local storage and session storage?

Answer : Some differences between local storage and session storage :
1. Normally,Local storage size is 5mb/10mb.And Session storage size is 5mb.(this amount can be change according to the browser)
2. Local storage data is available for any tab,But Session storage data is available only a particular tab.
3. Local storage data remains even after closing the browser,but Session storage is cleaned when closing the browser.
4. Session storage is working session based,but Local storage is not like this.
5. Session storage data is expiry.But Local storage data doesn't expire.
6. Local storage data has to be deleted manually,But Session storage data has to be deleted automatically.
